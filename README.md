# Word guessing game
This a word guessing game

```sh
g++ wordguessinggame.cpp -o wordguessinggame
./wordguessinggame
```


```txt
                INSTRUCTIONS
1.You shoud give the four letters word and word should be meaningful.
2.All the four letters in the word should be distinct.
3.Terminologies involved
     =>if GOAT=n, then n letter in your word is correct and lies in correct place.
     =>if COW=n, then n letters in your word are correct and lie in wrong place.
            eg.if your word is love and answer is wolf 
               goat=1  cow=1
4.Press 'q' to quit the game.
5.Press 'n' for enter into new game.
```
